# Personal Introduction Program

##  Project Description
This is a beginner Python project that asks the user for their personal information and displays a friendly introduction message.

##  Features
- Uses `input()` to collect user information
- Stores information in variables
- Displays formatted output using `print()` and f-strings
- Beginner friendly Python program

##  Technologies Used
- Python 3
